    04-Docker常用命令

    luffy 11/8/2023 docker

    # 启动类命令

    # 启动docker

    systemctl start docker
    1

    # 停止Docker

    systemctl stop docker
    1

    # 重启Docker

    systemctl restart docker
    1

    # 查看状态

    systemctl status docker
    1

    # 设置开机自启

    systemctl enable docker
    1

    # 帮助类命令

    # 查看Docker版本

    docker version
    1

    # 查看Docker概要信息

    docker info
    1

    # 查看Docker总体帮助文档

    docker --help
    1

    # 查看docker具体命令帮助文档

    docker 具体命令 --help
    1

    # 镜像命令

    # 列出本地主机上的镜像

    docker images
    1

    参数：

    • -a：列出所有镜像（含历史镜像）
    • -q：只显示镜像ID
    • -f：过滤

    # 在远程仓库中搜索镜像

    docker search 镜像名称
    1

    参数：

    • -f：过滤
    • --limit 数量：只展示前几项

    # 下载镜像

    docker pull 镜像名称[:tag]
    1

    # 查看占据的空间

    docker system df
    1

    # 删除镜像

    docker rmi 镜像名称/ID
    1

    可以使用空格分隔，删除多个镜像：

    docker rmi 镜像1 镜像2 镜像3
    1

    删除全部镜像：

    docker rmi -f $(docker images -qa)
    1

    # 虚悬镜像

    仓库名、标签都是<none>的镜像，俗称虚悬镜像（dangling image）。

    # 命令自动补全

    # 如果镜像中有ubuntu，查看输入ub按下tab是否可以补全
docker run ub

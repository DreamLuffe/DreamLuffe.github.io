    拉取远程仓库其他分支到本地

    luffy 4/10/2023 Git

    # 已经有仓库，拉取远程仓库其他分支到本地

    在编写其他分支的是时候，为了不污染新的分支，拉去远程仓库新建分支

    //已经有仓库，拉取远程仓库其他分支到本地
git checkout -b 新的分支名 远程仓库别名/远程仓库分支名

案例：git checkout -b waterCondition origin/waterCondition

//查看远程已经本地分支
git branch -a
